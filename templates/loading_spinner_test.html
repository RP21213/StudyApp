Usage Instructions

How to Use the Loading Spinner:

  1. Include the loading-spinner.js script in your HTML
  2. Add the CSS styles for .loading-spinner and .button-loading
  3. Create a LoadingSpinner instance: new LoadingSpinner('button-id')
  4. Call showLoading() to start loading
  5. Call showSuccess() or showError() to finish

Features:

  • Prevents multiple clicks during loading
  • Customizable loading, success, and error text
  • Automatic button state management
  • Reusable across different buttons
  • Callback support for success actions
